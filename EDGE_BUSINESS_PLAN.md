# Edge 배포 검토용 비즈니스/서비스 개선안

## 1) 현재 제품 포지셔닝 요약
- 현재 확장 기능은 `Alt+Q`로 현재 영상 URL을 기반으로 Gemini 요약 프롬프트를 자동 전송하는 생산성 툴입니다.
- 지원 플랫폼은 YouTube/Vimeo/Twitch/Dailymotion 중심으로 되어 있어, 학습/정보성 영상 소비자에게 맞는 니치 제품입니다.

## 2) 수익화 모델 제안 (우선순위 순)

### A. Freemium + Pro 구독 (가장 현실적)
- **무료 플랜**
  - 일/월 요약 횟수 제한
  - 기본 요약 템플릿만 제공
  - 단일 언어 출력
- **Pro 플랜 (월 구독)**
  - 요약 무제한 또는 고한도
  - 템플릿(시험 대비, 회의록형, 블로그 초안형, 학습카드형)
  - 다국어 동시 출력 (예: 한국어 + 영어)
  - 개인 노트앱(Notion/Obsidian/Google Docs) 내보내기
  - 히스토리 저장/검색

**가격 가이드(초기 가설)**
- Lite: $3.99 / month
- Pro: $7.99 / month
- Team(3~10석): 좌석당 $4.99 / month

### B. B2B2C 번들 (교육기관/인강 커뮤니티)
- 온라인 강의 수강생 전용 번들 제공
- 학원/교육 플랫폼에 화이트라벨 위젯 형태 제공
- 라이선스 단가: 월 고정 + MAU 기반 과금

### C. 제휴/어필리에이트
- 영상 학습 SaaS(노트앱, 문제은행, 언어학습)와 제휴
- 요약 후 CTA 카드 삽입 (과도한 광고 대신 맥락형 추천)

## 3) 핵심 KPI 설계

### 북극성 지표 (North Star)
- **주간 유효 요약 완료 사용자 수 (WAU with successful summary submit)**

### 퍼널 KPI
1. 설치 → 첫 요약 실행률
2. 첫 요약 → 7일 리텐션
3. 무료 사용자 → 유료 전환율
4. 유료 해지율(월)

### 품질 KPI
- 자동 입력 성공률
- 평균 처리 시간(단축키 → 전송 완료)
- 지원 URL 실패율
- 사용자 체감 품질 점수(요약 만족도 5점)

## 4) 서비스 개선안 (제품 측면)

### 4-1. 즉시 적용 (2~4주)
1. **요약 템플릿 프리셋**
   - “시험 대비”, “실무 요점”, “숏폼용 3줄 요약”
2. **출력 길이/깊이 옵션**
   - 1분 요약 / 5분 요약 / 상세 요약
3. **결과 저장 및 재열람 히스토리**
   - 현재 코드에 `summary.html`/`summary.js` 구조가 있어 확장 가능성이 큼
4. **오류 가이드 UX 강화**
   - 자동 입력 실패 시 복사 버튼/재시도 버튼/원인 가이드

### 4-2. 중기 (1~2분기)
1. **클라우드 동기화 계정**
   - 브라우저/기기 간 히스토리 공유
2. **협업 기능**
   - 공유 링크, 팀 워크스페이스, 주석
3. **멀티 AI 백엔드**
   - Gemini 외 OpenAI/Claude 선택형
4. **학습형 기능**
   - 반복 출제, 퀴즈 자동 생성, 복습 알림

## 5) Edge 배포 관점 체크리스트

1. **권한 최소화**
   - 현재 `<all_urls>`는 심사/신뢰 측면에서 불리할 수 있어, 지원 사이트 중심으로 축소 권장
2. **개인정보/데이터 정책 문서화**
   - 어떤 데이터를 저장/전송하는지 명확히 고지
3. **브랜드/스토어 페이지 최적화**
   - 3문장 가치제안 + 실제 사용 GIF + before/after 예시
4. **초기 리뷰 수집 캠페인**
   - 설치 3회 사용 사용자에게 자연스러운 리뷰 요청

## 6) 90일 실행 로드맵(추천)

### Day 0~30
- 요약 템플릿, 길이 옵션, 오류 UX 개선
- 이벤트 로그(설치/요약성공/실패/재시도) 설계
- Edge 스토어 등록 + 랜딩 페이지 오픈

### Day 31~60
- 히스토리 기능과 내보내기(Notion/Docs) 추가
- 프리미엄 플랜 페이월 실험(A/B)
- 크리에이터/스터디 커뮤니티 1차 제휴

### Day 61~90
- 유료 전환 최적화(체험판, 할인, 연간 플랜)
- 팀 플랜 베타
- B2B 파일럿 2~3곳 운영

## 7) 리스크 및 대응
- **Gemini UI 변경 리스크**: 선택자 다변화 + 헬스체크 + 빠른 핫픽스 프로세스
- **플랫폼 정책 리스크**: 최소 권한/투명한 정책/명확한 사용자 동의
- **수익화 저항**: 무료 구간의 명확한 가치 유지 + Pro 차별 기능 강화

## 8) 결론
- 이 제품은 “영상 학습 생산성”이라는 명확한 사용 맥락이 있어, Freemium 구독 모델이 가장 자연스럽습니다.
- 단기적으로는 **성공률(기술 안정성)**과 **재방문 이유(히스토리/템플릿)**를 먼저 만들고,
  중기적으로 **내보내기/협업/팀 요금제**로 ARPU를 높이는 전략이 적합합니다.
